# Example Source Dataframe

An example source data table with top-level (pre-fractionation) metadata

## Usage

``` r
example_source_df
```

## Format

### `example_source_df`

A data frame with 15 rows and 5 columns:

- source:

  The unique name of the biological replicate or source_mat_id

- total_copies_per_g:

  The total amplicon copies as determined by qPCR

- total_dna:

  The total DNA isolated from the source

- Isotope:

  The isotope used in this source

- Moisture:

  An example of a relevant treatment
