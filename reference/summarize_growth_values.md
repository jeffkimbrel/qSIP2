# Summarize growth values

Summarize growth values

## Usage

``` r
summarize_growth_values(qsip_data_object, confidence = 0.9, quiet = FALSE)
```

## Arguments

- qsip_data_object:

  A qsip_data object

- confidence:

  (*numeric, default: 0.9*) The confidence level for the growth values

- quiet:

  (*logical, default: FALSE*) Suppress messages
