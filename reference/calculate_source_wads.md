# Calculate global weighted average density (WAD) value for a source_mat_id (internal)

Calculate global weighted average density (WAD) value for a
source_mat_id (internal)

## Usage

``` r
calculate_source_wads(sample_data)
```

## Arguments

- sample_data:

  (*qsip_sample_data*) Sample data object

## Value

A dataframe with two columns, 1) the source_mat_id and 2) the global WAD
value for that source_mat_id
