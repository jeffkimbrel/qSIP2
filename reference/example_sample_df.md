# Example Sample Dataframe

An example sample data table with fraction-related metadata

## Usage

``` r
example_sample_df
```

## Format

### `example_sample_df`

A data frame with 284 rows and 6 columns:

- sample:

  The unique sample id of the fraction

- source:

  The parent source id for this sample

- Fraction:

  The gradient position

- density_g_ml:

  The density of this fraction

- dna_conc:

  The concentration of DNA post-fractionation

- avg_16S_g_soil:

  qPCR amplicon count
