# Return the size of qsip data objects or each slot

Return the size of qsip data objects or each slot

## Usage

``` r
qsip_object_size(qsip_data_object, units = "auto")
```

## Arguments

- qsip_data_object:

  A qsip object or list of objects

- units:

  Units to format the size in. Default is "auto"
