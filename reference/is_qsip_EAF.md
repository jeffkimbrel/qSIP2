# Validate a qsip object has EAF values

Validate a qsip object has EAF values

## Usage

``` r
is_qsip_EAF(object, error = FALSE)
```

## Arguments

- object:

  The object to check if it is a qsip_data object with EAF calculations

- error:

  If TRUE it stops with an error message. If FALSE it doesn't error, but
  returns FALSE
