# Calculate N_Light_it

This is equation 3 from Koch, 2018

## Usage

``` r
calculate_N_light_it(N_total_it, M_heavy, M_labeled, M)
```

## Arguments

- N_total_it:

  The copy number of feature i at timepoint t

- M_heavy:

  The theoretical molecular weight of 100% labeled sequence

- M_labeled:

  The molecular weight of the labeled sequence

- M:

  The molecular weight of the unlabeled sequence
