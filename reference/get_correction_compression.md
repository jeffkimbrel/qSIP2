# Get compression values from a rlm correction

Get compression values from a rlm correction

## Usage

``` r
get_correction_compression(corrections)
```

## Arguments

- corrections:

  (*tibble*) A tibble of corrections from
  [`correct_gpd_rlm()`](https://jeffkimbrel.github.io/qSIP2/reference/correct_gpd_rlm.md)
