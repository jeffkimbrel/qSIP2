# Make a feature dataframe from a JGI coverage file

Make a feature dataframe from a JGI coverage file

## Usage

``` r
jgi_feature_df(coverage_file)
```

## Arguments

- coverage_file:

  A file path to a JGI coverage file

## Value

A dataframe suitable for `qSIP2` workflows
