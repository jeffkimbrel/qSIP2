# Show missing source_mat_ids and sample_ids

This function identifies shared and missing source_mat_ids and
sample_ids between the source, sample, and feature data objects.

## Usage

``` r
get_unshared_ids(qsip_data_object)
```

## Arguments

- qsip_data_object:

  (*qsip_data*) A qSIP data object

## Value

A message with unique IDs per category
