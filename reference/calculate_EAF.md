# Calculate EAF value (internal)

Calculate EAF value (internal)

## Usage

``` r
calculate_EAF(M_labeled, M, M_labeledmax, isotope)
```

## Arguments

- M_labeled:

  Molecular weight of the labeled feature

- M:

  Molecular weight of the unlabeled feature

- M_labeledmax:

  Theoretical molecular weight of a completely labeled feature

- isotope:

  The isotope to determine which calculation to run

## Value

EAF value
