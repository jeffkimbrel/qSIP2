# Validate a qsip object has been resampled

Validate a qsip object has been resampled

## Usage

``` r
is_qsip_resampled(object, error = FALSE)
```

## Arguments

- object:

  The object to check if it is a resampled qsip_data object

- error:

  If TRUE it stops with an error message. If FALSE it doesn't error, but
  returns FALSE
