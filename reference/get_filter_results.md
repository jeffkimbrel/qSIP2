# Tabular output summarizing filtering

Tabular output summarizing filtering

## Usage

``` r
get_filter_results(qsip_data_object, type = "counts")
```

## Arguments

- qsip_data_object:

  (*qsip_data*) An object of `qsip_data` class that has been filtered

- type:

  (*string, default: counts*) The type of output to return. Either
  "counts" or "feature_ids"
