# Return the original dataframe from a qsip_feature_data object

Return the original dataframe from a qsip_feature_data object

## Arguments

- x:

  (*qsip_feature_data*) A qSIP feature data object

- original_headers:

  (*logical*) Return the original column names

- type:

  (*source*, *sample* or *feature*) When giving a qSIP object the type
  determines which dataframe to retrieve
