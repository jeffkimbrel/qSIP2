# Calculate the WAD difference Z (internal)

This is just basic subtraction, so probably overkill to make a function!

## Usage

``` r
calculate_Z(labeled, unlabeled)
```

## Arguments

- labeled:

  (*string*) The column with the labeled WAD or mean labeled WAD value

- unlabeled:

  (*string*) The column with the unlabeled WAD or mean unlabeled WAD
  value

## Value

a value for the difference between labeled and unlabeled

## Details

This function corresponds to equation 4 from Hungate, 2015
