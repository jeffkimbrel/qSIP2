# Validate a qsip object has been run through growth workflow

Validate a qsip object has been run through growth workflow

## Usage

``` r
is_qsip_growth(object, error = FALSE)
```

## Arguments

- object:

  The object to check if it is a growth qsip_data object

- error:

  If TRUE it stops with an error message. If FALSE it doesn't error, but
  returns FALSE
