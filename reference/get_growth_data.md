# Get dataframe of resampled growth data

This function will return a dataframe of the resampled growth data
embedded in a `qsip_data` object.

## Usage

``` r
get_growth_data(qsip_data_object)
```

## Arguments

- qsip_data_object:

  A `qsip_data` object that has been run through
  [`run_growth_calculations()`](https://jeffkimbrel.github.io/qSIP2/reference/run_growth_calculations.md)

## Value

A dataframe of the resampled data
